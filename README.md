# Gestionnaire de cadeaux en Python

## Description
Projet Python en console qui permet de gérer une liste de cadeaux.
Fonctionnalités disponibles :
- Ajouter un cadeau
- Modifier un cadeau
- Supprimer un cadeau
- Lire la liste de cadeaux
- Sauvegarder / charger la liste de cadeaux dans un fichier JSON
- Quitter le programme

Ce projet est conçu comme un exercice pratique pour manipuler les listes, dictionnaires et fichiers JSON en python.

---

## Installation
Clonez le dépôt et placez-vous dans le dossier

## Lancement du script
lancer le fichier python : main.py

## Exemple d'utilisation
1- Ajouter un cadeau
2- Modifier un cadeau
3- Supprimer un cadeau
4- Afficher la liste
5- Sauvegarder la liste
6- Charger la liste
7- Quitter

## Structure du projet
. main.py : code principal du gestionnaire de cadeaux
. listedecadeaux.json : fichier de sauvagarde de la liste de cadeaux
. README.md : documentation du projet

## Technologies utilisées
. Python 3.X
. JSON (JavaScript Object Notation) pour la sauvegarde des données

## Exemple de contenu du fichier JSON
[
    {
        "1": {
            "Nom": "Montre",
            "Prix": "9.99€",
